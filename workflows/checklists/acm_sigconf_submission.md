# ACM SIGCONF Submission Checklist

- Confirm the target ACM event actually uses a double-blind proceedings track and the ACM `sigconf` template.
- Confirm anonymization is complete, including acknowledgements, PDF metadata, and repository links.
- Confirm the target event's page limit and appendix policy rather than assuming a generic ACM default.
- Confirm any supporting files are explicitly allowed and anonymized.
- Confirm ACM-specific publication metadata, rights, and classification requirements for the target event.
