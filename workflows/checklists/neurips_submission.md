# NeurIPS Submission Checklist

- Confirm the main paper body stays within the current NeurIPS content-page limit.
- Confirm the required paper checklist is included where the current CFP expects it.
- Confirm acknowledgements and other identifying metadata are omitted from the anonymized submission.
- Confirm appendices, supplementary files, and linked code preserve double-blind anonymity.
- Confirm any anonymous references to prior or concurrent work are provided in an allowed supplementary form.
