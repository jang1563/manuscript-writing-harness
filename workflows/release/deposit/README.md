# Deposit Notes

Archive-target notes for Zenodo, OSF, or other frozen-release deposits live here.

These notes are generated from the archive-export layer and summarize:

- intended deposit targets
- required frozen files
- checksum inventory location
- anonymization and provenance reminders before final deposit

Release-specific deposit metadata artifacts also live here:

- profile-scoped `CITATION.cff`
- `codemeta.json`
- Zenodo deposit metadata JSON
- OSF deposit metadata JSON
