# Figure Bundles

Tracked figure bundles live here.

A bundle is a named manuscript-facing figure set with:

- explicit recipe ancestry
- tracked manuscript fragments
- bundle-level review and validation outputs
- additive wiring into `manuscript/sections/03_results.md`

The two v1 exemplar bundles are:

- `bundle_bulk_omics_deg_exemplar`
- `bundle_ai_ml_evaluation_exemplar`

Use `python3 scripts/figures_cli.py list-bundles` to inspect tracked bundles and `python3 scripts/figures_cli.py apply-bundles --all` to refresh the managed Results bundle block.
