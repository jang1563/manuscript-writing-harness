# Reference Metadata Helpers

Use this directory for:

- `bibliography_source.yml`, which records the expected Zotero Better BibTeX auto-export contract for `references/library.bib` and whether the current export has been confirmed for the real manuscript
- DOI enrichment outputs
- PMID or PMCID mapping tables
- duplicate-resolution logs
- curated citation exceptions

Keep machine-generated metadata separate from the human-curated Zotero source of truth.
